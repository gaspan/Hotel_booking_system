# README

## Technology
- express js
- nodejs v8 linux
- Mysql
- Postman 

## SETTING YOUR OWN DATABASE
set it in folder config > database.js
create database in in your mysql :
```bash
mysql> CREATE DATABASE hotel_booking
```

## INSTALL DEPENDENCIES
```bash
$ npm install
$ npm install -g knex
$ knex migrate:latest
```

## RUN SERVER
inside repository
```bash
$ node server.js 
```
